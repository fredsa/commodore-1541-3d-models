# Commodore 1541 3D printer parts

3D printed parts for the [Commodore 1541 floppy disk drive](https://wikipedia.org/wiki/Commodore_1541).


## Face plate

1.  Print [Commodore 1541 Replacement Faceplate](https://www.thingiverse.com/thing:4810473)
    by [Edward Van Every](https://www.thingiverse.com/evanevery)


## 5mm LED insert

1.  Download and install [OpenSCAD](https://openscad.org/)

2.  Open `led-holder.scad`

3.  Press F6 to render the model

## Drive door latch

1.  Print the modified `modified-1541_alps_drivedoor.stl`.

    This model is based on [Commodore 1541 ALPS drive door latch replacement](https://www.printables.com/model/125620-commodore-1541-alps-drive-door-latch-replacement)
    by [@InsaneDruid](https://www.printables.com/@InsaneDruid) and has been modified as follows:

    - Moved rod 1.1mm inward to fix clearance issue when opening/closing latch.
